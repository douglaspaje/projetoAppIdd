GiveMyCar

-> Usuário obtém o app.
-> Primeira tela, um tour com dicas de como usar o app.
-> Tela principal, usuário escolhe entre adquirir um carro ou cadastrar seu carro para venda.
-> Escolhida a opção de adquirir, redireciona para a listagem de veículos.
  -> Possíveis filtros (faixa de preço, geo localização, ano, modelo )
  -> Ao clicar no carro envia uma notificação para o dono do anuncio.
-> Escolhida a opção de vender, redireciona para o cadastro de veículo.
  -> Cadastro de dados pessoais do vendedor
  -> Cadastro de um ou mais veículos
